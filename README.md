# Calculadora Tocha

Calculadora en Python con interfaz Tkinter — versiones básica y científica.

Rápido inicio:

- Ejecutar la app:

```bash
python main.py
```

- Ejecutar tests:

```bash
python -m unittest discover -v
```

- Instalar herramientas de desarrollo:

```bash
python -m pip install -r requirements.txt
```

Características:

- Interfaz Tkinter con operaciones básicas y científicas.
- Soporte de memoria (M+, M-, MR) con persistencia (`memory.json`).
- Botón `%` con semántica tipo calculadora (A + B% → A + A*B/100).
- Funciones científicas: sin/cos/tan, inversas, log, ln, exp, abs, factorial, raíz.
